
Tugas RESTful API menggunakan Python FastAPI dan Supabase sebagai backend database.

Persiapan

1. Buat project di Supabase dan buat table 
CREATE TABLE books (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  title TEXT NOT NULL,
  author TEXT NOT NULL,
  is_read BOOLEAN DEFAULT false,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT timezone('utc'::text, now()) NOT NULL
);

2. Salin nilai `SUPABASE_URL` dan `SUPABASE_KEY` dari dashboard Supabase ke file `.env`.

Instalasi

Pada folder proyek jalankan:

pip install fastapi uvicorn supabase python-dotenv

Menjalankan server

uvicorn main:app --reload
